Arquivos principais:
- index.html

Melhorias incluídas nesta versão:
1. Tema verde claro institucional.
2. Tipografia corrigida, com título mais legível e sem bloco gigante em caixa alta.
3. Bloco de acesso rápido para Cronograma, Relatórios, Avaliação e Inovação.
4. Painel de qualidade com quatro frentes: Atendimento, Processos, Tecnologia e Equipe.
5. Sistema de avaliação com estrelas, armazenamento local e opção para baixar CSV.
6. Barra de progresso de leitura no topo.
7. Botão flutuante para voltar ao topo.
8. Cards com sombras suaves, animações e microinterações.

Publicação:
- Envie todos os arquivos para Cloudflare Pages, Netlify, Vercel ou qualquer hospedagem estática.
- O arquivo inicial deve ser index.html.

Observação:
Os links dos PDFs ainda apontam para o endereço original do Lovable. Para deixar o site totalmente independente, coloque os PDFs dentro da pasta pdfs e edite os links no index.html.
